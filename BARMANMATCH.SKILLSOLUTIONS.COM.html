Intelligence layer

8 Agenti AI.
Sempre al lavoro.

Una suite di agenti specializzati opera in background su ogni processo critico — da Claude claude-sonnet-4-6 via API Anthropic, orchestrati con LangChain/LangGraph.

Agent 01
🔍
Onboarding Verifier

Verifica documenti e certificazioni dei nuovi lavoratori in automatico. Riduce il tempo di approvazione da 48h a meno di 2h.

Trigger: upload documento
Agent 02
⚖️
Matching Engine

Calcola il ranking dei lavoratori compatibili per ogni turno. Score composito su pertinenza (40%), qualità (35%), relazione (15%), reattività (10%).

Trigger: pubblicazione turno
Agent 03
🔎
Smart Search NLU

Converte ricerche in linguaggio naturale in query filtrate. "Cameriere esperto fine dining sabato sera zona Navigli" → filtri strutturati automatici.

Trigger: ricerca struttura
Agent 04
🚀
Profile Booster

Analizza il profilo del lavoratore e genera 3–5 suggerimenti concreti per migliorare visibilità e ranking. Impatto stimato quantificato per ogni azione.

Trigger: accesso profilo
Agent 05
⭐
Review Analyzer

Modera le recensioni post-turno prima della pubblicazione. Blocca contenuti inappropriati, estrae topic e sentiment per analytics interna.

Trigger: invio recensione
Agent 06
🛡️
Fraud & Trust Monitor

Monitora in background comportamenti anomali: review bombing, account farming, no-show sistematici, wage manipulation. Escalation automatica.

Trigger: schedulato h1 + real-time
Agent 07
💬
Support Chatbot

Gestisce il supporto in-app per lavoratori e strutture. RAG su knowledge base. Escalation automatica a operatori umani per casi complessi.

Trigger: apertura chat supporto
Agent 08
📈
Demand Forecaster

Prevede il fabbisogno di personale per città e ruoli nelle settimane successive. Alert di carenza proattivi e raccomandazioni recruiting.

Trigger: domenica sera + eventi
Flussi orchestrati

I workflow
multi-agente.

A
Onboarding nuovo lavoratore
→Form + documenti caricati
→Agent 01 verifica (APPROVED / MANUAL_REVIEW / REJECTED)
→Agent 04 genera suggerimenti profilo iniziali
→Agent 08 controlla domanda attiva in città
→Push: "Ci sono turni disponibili per te a [città]!"
B
Turno urgente pubblicato
→Struttura pubblica turno con flag urgente
→Agent 03 converte ricerca libera in filtri
→Agent 02 calcola ranking top-20
→Push a top-20 (ogni 5 min scende al successivo)
→Primo che accetta → conferma automatica + chat
C
Post-turno qualità e pagamento
→Check-out geolocalizzato → calcolo ore effettive
→Entrambi inseriscono recensione testuale
→Agent 05 modera (pubblicazione simultanea)
→Agent 06 check pattern rating in background
→Stripe Connect: pagamento entro 24h + Agent 04 update
